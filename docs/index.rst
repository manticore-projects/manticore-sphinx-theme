Key Features
------------

- **High throughput** — process millions of records per second
- **Pluggable backends** — H2, PostgreSQL, Oracle, and more
- **Zero-downtime deployments** via rolling upgrade support
- **Comprehensive monitoring** with Prometheus and Grafana dashboards
- **Dark mode** — automatic, based on your OS preference
